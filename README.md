iOS Developer
-------------

Since childhood, my connection with the computer has been strong. Despite my young age, I could install programs, games, and so on. More precisely, I could do everything I needed from the computer. But that's where my progress stopped. I wondered how all the programs in the computer work and how they are created, but I never received an answer. I did not know about programming until my 1st year of university, that is, until the age of 19. And only when I entered the university I got acquainted with C ++, then C #, JavaScript, and Python. Since I studied in the specialty "mathematical and computer modeling", the university mainly solved problems for differential equations. Different tasks of varying complexity trained the base in programming well. After graduating from the university, I faced the realities of life, such as the underdevelopment of my field "mathematical and computer modeling" in my country. With such a base and with a love for programming, I could not change my activity for something else. At that moment, I was very interested in creating applications, that is, mobile development. Then I chose to study and bought iOS developer courses on the Geek Brains platform. Need experience, salary is not important. Completed courses on the GeekBrains platform: - Fundamentals of the SWIFT language - iOS user interface - applications - Client-server iOS - applications - Performance of iOS applications - Architectures and design patterns on SWIFT - Framework SwiftUI Applications developed on the course: - VK client - Who wants to be a millionaire - XO Game
